CFF2–1: FDArray with 256 Elements

The font for this test case is FDArray Test 257, a test font by Adobe. The font has been engineered to map Unicode codepoints to a CFF FDArray with 256 elements. Each glyph shows the final two hexadecimal digits of the input codepoint. For a more detailed description please see Adobe’s README file.
